# Desafio de Projeto

O projeto consiste na inserção e transformação de dados  oriundos de um banco de dados criado no azure no Power BI.

Etapas realizadas:
--------

- Configurar o setup de banco de dados na azure.
  
  Problema enfrentado: não foi possivel criar o banco de dados no azure .
  
  Solução encontrada : criar um banco de dados utilizando o Sql instalado no computador.
  

- Utilizar o script fornecido, adaptando para o Sql.

- Integrar o banco de dados do Sql com Power BI.

- Realizar as transformações:

Verificação de cabeçalho e tipos de dados, modificação dos valores monetarios, remoção de valores nulos, mescla de consultas employee e departament, 
união da coluna nome e sobrenome em uma coluna, mescla os nomes departament e localização.

- criação do relatório.
